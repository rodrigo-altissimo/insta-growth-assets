# Insta Growth Assets

Public asset hosting for Instagram Graph API publications.

This repo hosts only image assets that need to be publicly accessible for the Meta Graph API to fetch and publish to Instagram. No source code, no strategy documents, no private content.

## Structure

```
posts/
└── {NNN}-{slug}/
    ├── slide-01.png
    ├── slide-02.png
    └── ...
```

Each folder corresponds to a published Instagram post. Slides are 1080x1350px (4:5 portrait), generated by the Insta Growth Machine pipeline.

## Usage

Raw URLs follow the pattern:

```
https://raw.githubusercontent.com/rodrigo-altissimo/insta-growth-assets/main/posts/{NNN}-{slug}/slide-{NN}.png
```

These URLs are used as `image_url` parameters in Instagram Graph API media container creation.
